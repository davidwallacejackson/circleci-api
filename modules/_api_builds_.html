    Get recent build summaries for a project

    Supported query parameters:

    example

    limit - The number of builds to return. Maximum 100, defaults to 30.

    example

    offset - builds starting from this offset, defaults to 0.

    example

    filter -Restricts which builds are returned. Set to "completed", "successful", "failed", "running"
